## E-handel (2025)

**Stack:** Next.js, TypeScript, Tailwind, Supabase  

En komplett e-handelsplattform med modern teknik. Några funktioner:

- **Varukorg**: Sparar produkter för anonyma användare i databasen, kopplat till ett sessions-ID via cookies. Varukorgen överförs automatiskt till användar-ID vid inloggning.
- **Autentisering**: Byggt med Supabase Auth – användarkonto, lösenordsåterställning, orderhistorik och frivilliga personuppgifter.
- **Responsiv design** med Tailwind.
- Fler funktioner under utveckling.

[Se projektet live](https://e-commerce-next-js-nc.vercel.app)
